  - индикацию активных трейдеров

#### 1.2 Ручной режим в Sandbox
- [ ] Удалить из Sandbox элементы автотрейдера (если ещё остались)
- [ ] Добавить панель «Manual Order»:
  - выбор инструмента (Dropdown, можно использовать `selectedInstrument` или отдельный)
  - выбор типа ордера: Market / Limit (при Limit – поле цены)
  - выбор направления: BUY / SELL

- [ ] При отправке вызывать `OrderManager.processSignal` (или специальный метод `sendManualOrder`), который выставит ордер и, если заданы, стоп‑заявки/защитные лимитные ордера
