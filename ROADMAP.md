- Новые стратегии (6 шт.): InitialBalance, VABreakoutRetest, SFP, AnchoredVWAP, Absorption, Exhaustion
- Регистрация всех стратегий в фабрике

- Трейлинг‑стоп по волатильности (адаптивный)

- [c] **Передача конфигурации сетки на сервер** вместе с параметрами batch'а (эндпоинт `POST /api/backtest/batch` расширяется)

### 4. Учёт фаз рынка при прогонах
_Цель: чтобы сервер автоматически определял фазу рынка для каждого инструмента и периода, и группировал результаты._
