# 🚀 WeToTheMoon – Дорожная карта (ROADMAP)

Дата обновления: 2026-06-12

## ✅ Завершённые этапы

### Инфраструктура и основа
- gRPC‑сервисы T‑Invest, marketDataBus, HistoricalDataLoader
- VolumeProfileEngine с распределением объёма
- Базовые стратегии (7 шт.)
- BacktestEngine с VirtualPortfolio
- Batch‑бэктест v2 (работает идентично одиночному)

### Скринер инструментов
- ScreenerService с фильтрацией по VA и POC
- UI‑вкладка «Скринер» с быстрым запуском бэктеста

### Стабилизация
- Автоматическое переподключение gRPC‑стримов
- Мониторинг памяти

### Оптимизация UI
- Компактный режим (скрытие второстепенных панелей)
- Перенос управления стримом в верхнюю панель

### Облачный бэктест (базовый)
- Перенос вычислительного ядра на сервер (все модули)
- Express‑сервер с эндпоинтами `/api/backtest/tasks`, статуса и результатов
- BacktestQueue с последовательным выполнением
- Развёртывание на облачной ВМ (Cloud.ru), настройка сети и файрвола
- Тестирование API (curl‑скрипт)
- Клиентская интеграция: IPC‑каналы, preload, UI‑вкладка «Cloud»

### Методология Trader Dale (полная реализация)
- MarketPhaseDetector – определение рыночных фаз (Balance, Trend, Breakout, Chop)
- OrderFlowEngine – расчёт дельты, кумулятивной дельты, детекция абсорбции, истощения, айсберг‑заявок
- CompositeProfileService – накопление дневных профилей, свинг‑уровни
- StrategyManager – автоматический выбор стратегии по фазе рынка
- Новые стратегии (6 шт.): InitialBalance, VABreakoutRetest, SFP, AnchoredVWAP, Absorption, Exhaustion
- Регистрация всех стратегий в фабрике

### Улучшенное управление рисками
- Динамический размер позиции по ATR
- Трейлинг‑стоп по волатильности (адаптивный)
- Интеграция с OrderFlow для корректировки стопов в реальном времени

### Облачный бэктест-фермер (24/7 массовое тестирование) – базовый функционал
- [x] Серверная инфраструктура (Supabase, BacktestQueue, batch‑эндпоинты)

- [x] Автономность – клиент может отключиться, задачи продолжаются; результаты сохраняются в Supabase
- [x] Поддержка всех стратегий (13 шт.) и параметров риск‑менеджмента (стоп‑лосс, тейк‑профит, трейлинг, динамический лот)
- [x] Унификация загрузки свечей:
  - цикл по дням в локальном `HistoricalDataLoader` (уже было) и в облачном (исправлено)
