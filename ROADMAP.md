# 🚀 WeToTheMoon – Дорожная карта (ROADMAP)

Дата обновления: 2026-06-07

## ✅ Завершённые этапы

### Инфраструктура
- gRPC-сервисы T‑Invest (MarketData, Orders, Operations и др.)
- Шина marketDataBus (StreamCandle, StreamTrade)
- Исторический загрузчик (HistoricalDataLoader)
- Подключение к песочнице и real-time потокам
- IPC-обработчики для торгового ассистента

### Объёмный профиль
- VolumeProfileEngine (POC, VA, HVN/LVN, сигналы POC_BREAKOUT и др.)
- Нормальное распределение объёма внутри свечи (resolution, spreadFactor)
- Кэширование профилей, предотвращение дублирующихся сигналов
- Базовые стратегии (Trend, TrendPro, POC Pullback, Daily VA Reversal, FVG Volume, Rejection, Volume Accumulation)

### Бэктестинг
- BacktestEngine с VirtualPortfolio
- Batch‑бэктест v2 (с точным повторением логики одиночного бэктеста) — **актуальная версия, выбрана по умолчанию**
- Экспорт результатов в JSON через UI

### Скринер инструментов
- ScreenerService с фильтрами по объёму, ширине VA, силе POC
- Использование часовых свечей за 2 дня для точного профиля
- UI‑вкладка «Скринер» с таблицей подходящих бумаг и возможностью быстрого запуска бэктеста

### Стабилизация
- Автоматическое переподключение gRPC‑стримов (exponential backoff)
- Мониторинг памяти (process.memoryUsage) и передача метрик в UI
